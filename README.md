# Projet Voltaire vocabulaire
Une aide pour le vocabulaire du projet voltaire. 


## Usage/Examples

### Comment ça marche?
* [1] Rentrer le mot de vocabulaire a connaitre.
    * exemple : `Charisme`
* [2] La définition apparait
    * exemple : `Charisme = Qualité d'une personnalité fascinante`
